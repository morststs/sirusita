## bash コマンド

Linux/macOS のシェル（bash）でよく使う基本コマンドとシェルスクリプトの文法をまとめたチートシート。ファイル操作から検索、権限、ループ、便利なワンライナーまでを一覧化。

---

## 1. ファイル・ディレクトリ操作

* **`ls -la`**：隠しファイルを含めた一覧を詳細（権限・サイズ・更新日時）付きで表示
* **`ls -lh`**：ファイルサイズを人間が読みやすい単位（K/M/G）で表示
* **`cd -`**：直前にいたディレクトリへ戻る
* **`pwd`**：現在のディレクトリの絶対パスを表示
* **`mkdir -p a/b/c`**：途中の階層も含めてディレクトリを一括作成
* **`cp -r <元> <先>`**：ディレクトリを再帰的にコピー
* **`cp -a <元> <先>`**：属性（権限・タイムスタンプ・シンボリックリンク）を保持してコピー
* **`mv <元> <先>`**：ファイルの移動またはリネーム
* **`rm -rf <対象>`**：ディレクトリを中身ごと強制削除（取り消し不可なので注意）
* **`touch <ファイル>`**：空ファイルの作成、または既存ファイルの更新日時を現在に変更
* **`ln -s <実体> <リンク名>`**：シンボリックリンク（ショートカット）を作成

```bash
# 拡張子 .log のファイルだけをまとめて backup ディレクトリへコピー
mkdir -p backup
cp -a *.log backup/

# 一時ファイルを作成してすぐ削除
touch tmp.txt && rm -f tmp.txt
```

## 2. ファイル内容の表示・確認

* **`cat <ファイル>`**：ファイルの内容を丸ごと表示
* **`less <ファイル>`**：ページ単位でスクロール表示（`q` で終了、`/語` で検索）
* **`head -n 20 <ファイル>`**：先頭から 20 行を表示
* **`tail -n 20 <ファイル>`**：末尾から 20 行を表示
* **`tail -f <ファイル>`**：追記をリアルタイムに追従表示（ログ監視に便利）
* **`wc -l <ファイル>`**：行数をカウント
* **`diff <A> <B>`**：2 つのファイルの差分を表示

## 3. 検索（find / grep）

* **`find . -name "*.txt"`**：カレント以下から名前が一致するファイルを再帰検索
* **`find . -type d -name "node_modules"`**：ディレクトリ（`-type d`）のみを対象に検索
* **`find . -mtime -1`**：過去 1 日以内に更新されたファイルを検索
* **`find . -size +10M`**：10MB を超えるファイルを検索
* **`grep -rn "TODO" .`**：カレント以下を再帰（`-r`）し行番号付き（`-n`）で文字列検索
* **`grep -i "error" app.log`**：大文字小文字を無視して検索
* **`grep -v "debug" app.log`**：マッチしない行のみを抽出（除外）
* **`grep -E "foo|bar" <ファイル>`**：拡張正規表現で複数パターンを検索

```bash
# .js ファイルから "console.log" を含む行を、該当ファイルと行番号付きで一覧
grep -rn --include="*.js" "console.log" .

# 30 日より前に更新された .tmp ファイルを検索して削除
find /tmp -name "*.tmp" -mtime +30 -delete
```

## 4. パイプ・リダイレクト

* **`<コマンド> | <コマンド>`**：左の出力を右の入力へ繋ぐ（パイプ）
* **`> <ファイル>`**：標準出力をファイルへ上書き保存
* **`>> <ファイル>`**：標準出力をファイルへ追記
* **`2> <ファイル>`**：標準エラー出力をファイルへリダイレクト
* **`2>&1`**：標準エラーを標準出力へ統合（`> out.log 2>&1` でまとめて保存）
* **`< <ファイル>`**：ファイルを標準入力として渡す
* **`tee <ファイル>`**：標準出力を画面とファイルの両方へ同時出力

```bash
# 実行結果とエラーをまとめてログに保存しつつ画面にも表示
./build.sh > build.log 2>&1
./build.sh 2>&1 | tee build.log

# プロセス一覧から特定プロセスを数える
ps aux | grep nginx | grep -v grep | wc -l
```

## 5. 権限とユーザー（chmod / chown）

* **`chmod 755 <ファイル>`**：所有者に rwx、グループ・その他に rx を付与（数値指定）
* **`chmod +x <ファイル>`**：実行権限を追加（シェルスクリプトの実行に必要）
* **`chmod -R 644 <ディレクトリ>`**：再帰的に権限を変更
* **`chown user:group <ファイル>`**：所有者とグループを変更
* **`chown -R user <ディレクトリ>`**：再帰的に所有者を変更
* **`umask 022`**：新規作成ファイルのデフォルト権限を制御するマスクを設定

> 数値権限の読み方: r=4, w=2, x=1 の合計。`755` = rwx(7) / r-x(5) / r-x(5)。

## 6. 変数・条件・ループ（スクリプト基礎）

* **`name="値"`**：変数への代入（`=` の前後にスペースを入れない）
* **`echo "$name"`**：変数の展開（ダブルクォート内で `$変数` が展開される）
* **`$(コマンド)`**：コマンド置換。コマンドの出力を文字列として埋め込む
* **`if [ "$x" = "a" ]; then ... fi`**：条件分岐（`[ ]` 内の前後スペース必須）
* **`for i in 1 2 3; do ... done`**：リストを繰り返すループ
* **`while read line; do ... done < file`**：ファイルを 1 行ずつ処理
* **`case "$x" in a) ...;; *) ...;; esac`**：パターンによる多分岐

```bash
#!/usr/bin/env bash
set -euo pipefail   # エラー時に停止・未定義変数禁止・パイプ失敗検知

# カレント直下の .png をまとめてリネーム
for f in *.png; do
  mv "$f" "img_$f"
done

# 数値比較（-eq / -lt / -gt など）
count=$(ls | wc -l)
if [ "$count" -gt 100 ]; then
  echo "ファイルが多すぎます: $count"
fi
```

## 7. xargs と複数コマンド連結

* **`<コマンド> | xargs <コマンド>`**：標準入力を引数として後続コマンドへ渡す
* **`xargs -n 1`**：引数を 1 つずつに分割して実行
* **`xargs -I {} <コマンド> {}`**：プレースホルダ `{}` を使って任意位置に引数を埋め込む
* **`find ... -print0 | xargs -0`**：NUL 区切りで空白を含むファイル名も安全に処理
* **`<コマンドA> && <コマンドB>`**：A が成功（終了コード 0）したときだけ B を実行
* **`<コマンドA> || <コマンドB>`**：A が失敗したときだけ B を実行
* **`<コマンドA>; <コマンドB>`**：成否に関わらず順番に実行

```bash
# 空白を含むファイル名も安全に一括削除
find . -name "*.bak" -print0 | xargs -0 rm -f

# 各行をプレースホルダで受けてリネーム
ls *.txt | xargs -I {} mv {} {}.bak
```

## 8. プロセス管理と便利ワンライナー

* **`ps aux`**：実行中の全プロセスを詳細表示
* **`top` / `htop`**：CPU・メモリ使用率をリアルタイム表示
* **`kill <PID>`**：指定 PID のプロセスへ終了シグナルを送信
* **`kill -9 <PID>`**：強制終了（SIGKILL）
* **`pkill <名前>`** / **`killall <名前>`**：名前でプロセスをまとめて終了
* **`jobs` / `fg` / `bg`**：ジョブの一覧・前面復帰・背面実行
* **`<コマンド> &`**：コマンドをバックグラウンドで実行
* **`nohup <コマンド> &`**：ログアウト後も実行を継続

```bash
# ポート 8080 を使っているプロセスを特定して終了
lsof -i :8080
kill -9 $(lsof -t -i :8080)

# ディスク使用量の大きいディレクトリ TOP10 を表示
du -sh ./* | sort -rh | head -n 10

# 重複行を除いて出現回数の多い順に並べる
sort access.log | uniq -c | sort -rn | head
```
