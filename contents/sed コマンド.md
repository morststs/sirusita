## sed コマンド

ストリームエディタ sed の使い方をまとめたチートシート。テキストの置換・削除・抽出を 1 行で行え、ファイルやパイプ入力をその場で加工できる。

---

## 1. 基本の置換（s///）

* **`sed 's/old/new/' <ファイル>`**：各行で最初に見つかった `old` を `new` に置換
* **`sed 's/old/new/g' <ファイル>`**：行内のすべての `old` を置換（g = global）
* **`sed 's/old/new/2' <ファイル>`**：行内の 2 番目の一致だけを置換
* **`echo "abc" | sed 's/b/X/'`**：パイプ入力に対して置換を適用
* **`sed 's#/path#/new#g' <ファイル>`**：区切り文字を `#` に変更（スラッシュを含む文字列に便利）

```bash
# ファイルの "foo" をすべて "bar" に置換して画面表示
sed 's/foo/bar/g' config.txt

# パスの置換は区切りを変えると読みやすい
echo "/usr/local/bin" | sed 's#/usr/local#/opt#'
```

---

## 2. フラグ（g / i / p）

* **`s/old/new/g`**：行内すべてを置換（global）
* **`s/old/new/i`**：大文字小文字を区別せずにマッチ（ignore case）
* **`s/old/new/p`**：置換が行われた行を出力（`-n` と併用が定番）
* **`s/old/new/gi`**：フラグは組み合わせ可能（全置換かつ大小無視）
* **`sed -n 's/old/new/p' <ファイル>`**：置換した行だけを抽出表示

```bash
# Error / ERROR / error を大小区別なくすべて [ERR] へ
sed 's/error/[ERR]/gi' app.log

# 置換が起きた行だけを表示
sed -n 's/TODO/DONE/p' notes.txt
```

---

## 3. 行の指定とアドレス

* **`sed -n '5p' <ファイル>`**：5 行目だけを表示
* **`sed -n '2,4p' <ファイル>`**：2〜4 行目を表示
* **`sed '$d' <ファイル>`**：最終行（`$`）を削除
* **`sed '3s/old/new/' <ファイル>`**：3 行目に限定して置換
* **`sed '/pattern/s/old/new/' <ファイル>`**：`pattern` を含む行のみ置換
* **`sed '1,10s/old/new/g' <ファイル>`**：1〜10 行目の範囲で置換
* **`sed '0~2p' <ファイル>`**（GNU sed）：1 行おき（偶数行）に処理

```bash
# 10 行目から末尾までの "v1" を "v2" に置換
sed '10,$s/v1/v2/g' versions.txt

# "BEGIN" を含む行から "END" を含む行までを表示
sed -n '/BEGIN/,/END/p' data.txt
```

---

## 4. 削除（d）と抽出（-n / p）

* **`sed 'd' <ファイル>`**：全行を削除（出力なし）
* **`sed '3d' <ファイル>`**：3 行目を削除
* **`sed '2,5d' <ファイル>`**：2〜5 行目を削除
* **`sed '/pattern/d' <ファイル>`**：パターンに一致する行を削除
* **`sed '/^$/d' <ファイル>`**：空行を削除
* **`sed '/^#/d' <ファイル>`**：`#` で始まるコメント行を削除
* **`sed -n '/pattern/p' <ファイル>`**：`-n`（自動出力抑止）+ `p` で grep 相当の抽出

```bash
# 設定ファイルからコメント行と空行を取り除く
sed -e '/^#/d' -e '/^$/d' config.ini

# "ERROR" を含む行だけを抽出（grep の代替）
sed -n '/ERROR/p' app.log
```

---

## 5. 複数コマンド（-e / セミコロン）

* **`sed -e 'cmd1' -e 'cmd2' <ファイル>`**：`-e` で複数の編集コマンドを並べる
* **`sed 'cmd1; cmd2' <ファイル>`**：セミコロンで複数コマンドを連結
* **`sed -f script.sed <ファイル>`**：コマンドをファイルから読み込んで実行
* **`sed '{ s/a/A/; s/b/B/ }' <ファイル>`**：波括弧でブロック化して複数処理

```bash
# 複数の置換をまとめて適用
sed -e 's/foo/bar/g' -e 's/baz/qux/g' input.txt
sed 's/foo/bar/g; s/baz/qux/g' input.txt
```

---

## 6. インプレース編集（-i）

* **`sed -i 's/old/new/g' <ファイル>`**：ファイルを直接書き換え（出力ではなく上書き）
* **`sed -i.bak 's/old/new/g' <ファイル>`**：`.bak` バックアップを残して書き換え
* **`sed -i '/pattern/d' <ファイル>`**：一致行を削除して上書き
* **`sed -i '$a 末尾行' <ファイル>`**：ファイル末尾に行を追記して上書き

```bash
# 複数ファイルのバージョン番号を一括置換（バックアップ付き）
sed -i.bak 's/1\.0\.0/1\.1\.0/g' ./*.json

# macOS（BSD sed）では拡張子の指定が必須（空でも -i '' とする）
sed -i '' 's/old/new/g' file.txt
```

> 注意: GNU sed は `-i`（拡張子なし可）、BSD sed（macOS）は `-i ''` のように空引数が必要。

---

## 7. 正規表現とキャプチャ（& / \1）

* **`sed -E 's/regex/new/' <ファイル>`**：拡張正規表現を使用（`+ ? ( ) |` をエスケープ不要に）
* **`sed 's/.*/[&]/' <ファイル>`**：`&` はマッチ全体を参照（行全体を `[]` で囲む）
* **`sed -E 's/(foo)(bar)/\2\1/' <ファイル>`**：`\1 \2` でキャプチャグループを参照・並べ替え
* **`sed -E 's/^([0-9]+).*/\1/' <ファイル>`**：行頭の数値だけを取り出す
* **`sed 's/^[ \t]*//' <ファイル>`**：行頭の空白・タブを削除（トリム）
* **`sed 's/[ \t]*$//' <ファイル>`**：行末の空白を削除

```bash
# 日付の区切りを / から - へ並べ替え（YYYY/MM/DD → YYYY-MM-DD）
echo "2026/06/28" | sed -E 's#([0-9]{4})/([0-9]{2})/([0-9]{2})#\1-\2-\3#'

# 各行を引用符で囲む
sed 's/.*/"&"/' list.txt
```

---

## 8. 応用ワンライナー

* **`sed -n '$=' <ファイル>`**：総行数を表示（`wc -l` 相当）
* **`sed '1d' <ファイル>`**：先頭行（CSV ヘッダなど）を除去
* **`sed 'G' <ファイル>`**：各行のあとに空行を挿入（行間を空ける）
* **`sed '/^$/N;/^\n$/D' <ファイル>`**：連続する空行を 1 行に圧縮
* **`sed -n '1~2p' <ファイル>`**（GNU sed）：奇数行だけを表示
* **`sed '1!G;h;$!d' <ファイル>`**：行の並びを逆順にする（tac 相当）
* **`sed 's/\r$//' <ファイル>`**：Windows 改行（CRLF）を LF に変換

```bash
# CSV のヘッダ行を除いて本文だけを処理
sed '1d' data.csv

# 行頭・行末の空白を同時にトリムし、空行は削除
sed -E 's/^[ \t]+|[ \t]+$//g; /^$/d' messy.txt
```
