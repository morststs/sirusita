## 1. 基本

- **/init** — リポジトリ内のファイルをスキャンして、プロジェクト用の `CLAUDE.md`（毎回読み込まれる常設メモ）のたたき台を生成。
- **/config** — 設定画面を開く（`/settings` も同義）。モデルや権限などの各種設定を確認・変更。
- **/model** — 使うモデルを切り替え（fable / opus / sonnet / haiku など）。ピッカー上で ← → を押すとeffortも調整可能。
- **/effort** — 推論の深さ（消費トークン量）を設定。`low / medium / high / xhigh / max` の段階で指定。
- **/clear** — 会話履歴をリセットして新規セッションを開始。CLAUDE.mdなどのプロジェクトメモは保持され、ファイル編集も残る。
- **/exit** — セッションを終了（`Ctrl+D` でも可）。

## 2. セッション管理

- **/compact** — 会話を要約して圧縮し、コンテキスト（トークン枠）を空ける。`/compact <重点>` で要約の焦点を指定可能。/clearと違い、それまでの文脈の流れは要約として保持される。
- **/memory** — `CLAUDE.md` の編集、自動メモ（auto-memory）のオン/オフ切り替え、登録内容の確認。
- **/resume** — 過去のセッションをID名または名前で再開（指定なしならピッカー表示）。
- **/rewind** — チェックポイントを使った巻き戻し（`Esc Esc` と同じ）。コードのみ／会話のみ／両方を選んで以前の状態に復元。※bashコマンドによるファイル変更や外部での手動編集は追跡対象外。
- **/rename** — 現在のセッションに名前を付ける/変更する。あとで `/resume` で見つけやすくするための機能。

## 3. 拡張・応用

- **/btw** — 「ところで」的な脇道の質問を、本筋の会話履歴を膨らませずに尋ねる（by the way の略）。
- **/voice** — プッシュトゥトークの音声入力をオン/オフ。
- **/mcp** — MCPサーバー（外部ツール連携）の管理とOAuth認証。GitHubやNotionなど外部サービスへの接続を扱う。
- **/schedule** — cronのような定期実行のクラウドエージェント（スケジュールタスク）を作成。
- **/remote-control** — セッションをリモート操作する機能。手元の端末以外（例：外出先）から作業を続けるための仕組み。
