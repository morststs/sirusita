## 1. 概要一覧（クイックリファレンス）

詳細な書き方を見る前の、サッと確認したいとき用の記法一覧。

| 要素 | 記述のイメージ |
| --- | --- |
| **見出し** | `# 見出し1` 〜 `###### 見出し6` |
| **太字・斜体** | `**太字**` / `*斜体*` |
| **箇条書きリスト** | `* 項目` または `- 項目` |
| **番号付きリスト** | `1. 項目1` `2. 項目2` |
| **チェックボックス** | `- [ ] 未完了` `- [x] 完了` |
| **引用** | `> 引用文` |
| **リンク** | `[表示テキスト](URL)` |
| **画像埋め込み** | `![代替テキスト](画像URL)` |
| **インラインコード** | ``code``（バッククォートで囲む） |
| **コードブロック** | ```トリプルバッククォート で上下を囲む |
| **水平線** | `---`（3つ以上のハイフンなど） |

## 2. マークダウン記述例サンプル集

実際の表示結果（レンダリング後）と、その元となるマークダウンの記述サンプル。

### 見出し（Headers）

# 見出し1

## 見出し2

### 見出し3

```markdown
# 見出し1
## 見出し2
### 見出し3

```

### テキストの装飾（Emphasis）

普通の間テキスト、**太字のテキスト**、*斜体のテキスト*、~~打ち消し線が入ったテキスト~~。

```markdown
普通の間テキスト、**太字のテキスト**、*斜体のテキスト*、~~打ち消し線が入ったテキスト~~。

```

### 箇条書きリスト（Unordered List）

* りんご
* みかん
* 温州みかん
* ポンカン


* バナナ

```markdown
* りんご
* みかん
    * 温州みかん
    * ポンカン
* バナナ

```

### 番号付きリスト（Ordered List）

1. 準備をする
2. 実行する
3. 片付けをする

```markdown
1. 準備をする
2. 実行する
3. 片付けをする

```

### タスクリスト / チェックボックス（Task List）

* [x] 今日のタスクを完了する
* [ ] 明日の準備をする

```markdown
- [x] 今日のタスクを完了する
- [ ] 明日の準備をする

```

### 引用（Blockquotes）

> これは引用文である。
> 長文の引用や、他のソースからの情報を明示するときに使用する。
> > ネスト（階層化）した引用も可能。
> 
> 

```markdown
> これは引用文である。
> 長文の引用や、他のソースからの情報を明示するときに使用する。
>> ネスト（階層化）した引用も可能。

```

### リンクと画像（Links & Images）

[Googleのトップページへ](https://www.google.com)

```markdown
[Googleのトップページへ](https://www.google.com)

![サンプルのロゴ画像](https://via.placeholder.com/150)

```
