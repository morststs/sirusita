## 数式（Math Expressions）

論文や技術文書に向けた、LaTeX記法による美しい数式表示。

### 1. インライン数式

文章の途中に数式を埋め込むための形式。前後にドル記号を1つずつ配置。

アインシュタインの有名な数式は $E = mc^2$ である。

```markdown
アインシュタインの有名な数式は $E = mc^2$ である。

```

### 2. ディスプレイ数式（独立行）

数式を中央揃えにして、独立した行として大きく強調表示する形式。前後にドル記号を2つずつ配置。

$$f(x) = \int_{-\infty}^{\infty} \hat{f}(\xi) e^{2\pi i x \xi} d\xi$$

```markdown
$$f(x) = \int_{-\infty}^{\infty} \hat{f}(\xi) e^{2\pi i x \xi} d\xi$$

```

### 3. 上付き文字・下付き文字

べき乗や添え字を表現するための基本記法。上付きは `^`、下付きは `_` を使用。

$x_{i}^{2} + y_{i}^{2} = z_{i}^{2}$

```markdown
$x_{i}^{2} + y_{i}^{2} = z_{i}^{2}$

```

### 4. 分数

複雑な分子・分母の関係を縦に並べて見やすく整理する記法。`\frac{分子}{分母}` の形で指定。

$$\frac{\partial f}{\partial x} = \lim_{\Delta x \to 0} \frac{f(x + \Delta x) - f(x)}{\Delta x}$$

```markdown
$$\frac{\partial f}{\partial x} = \lim_{\Delta x \to 0} \frac{f(x + \Delta x) - f(x)}{\Delta x}$$

```

### 5. 行列

数や記号を縦横の格子状に配置する記法。`matrix` や `pmatrix`（丸括弧付き）などの環境を使用。

$$A = \begin{pmatrix} a & b \\ c & d \end{pmatrix}$$

```markdown
$$A = \begin{pmatrix} a & b \\ c & d \end{pmatrix}$$

```
